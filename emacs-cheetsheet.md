Emacs チートシート (メモ)
=========================

カーソルの上下移動
--------------

### 上下

`[C-p]`
: 一行上へ

`[C-n]`
: 一行下へ

`[M-<]`
: ファイルの先頭へ

`[M->]`
: ファイルの末尾へ

### 左右

`[C-b]`
: 一文字左へ

`[C-f]`
: 一行右へ

`[C-a]`
: 行頭へ

`[C-e]`
: 行末へ


文字の削除
--------

### カーソルの文字削除

```
[C-d]
```

### カーソル位置から行末まで削除

```
[C-k]
```

### バックスペース

```
[C-h]
```

コピー＆ペースト
-------------

### コピー/カット

`[C-SPC]`で選択開始。`[M-w]`でコピー、`[C-w]`でカット。

```
[C-SPC]コピーするテキスト[M-w/C-w]
```

### 貼り付け

```
[C-y]
```


バッファー操作
-----------

### バッファの切り替え

```
[C-x b]バッファ名
```

### バッファの一覧

```
[C-x C-b]
```

### バッファを画面いっぱいに広げる

現在選択しているバッファをウィンドウいっぱいに広げる。

```
[C-x 1]
```

### バッファを消去する

```
[C-x k]
```

### バッファを上下に2分割

```
[C-x 2]
```

### バッファを左右に2分割

```
[C-x 3]
```
